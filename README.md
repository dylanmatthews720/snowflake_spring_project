# Snowflake Spring Forward

This repository contains materials for the **Snowflake Spring Forward AI Project**, focused on building an AI agent for conversational analytics using Snowflake Cortex tools.

## Contents

- **Snowflake Academia – Spring Project in an Hour.pdf**

This document walks through the process of building a **Snowflake Intelligence AI agent** that can answer questions by reasoning over both **structured healthcare data** and **unstructured biomedical research data**. :contentReference[oaicite:0]{index=0}

## Project Overview

The project demonstrates how to build an enterprise AI assistant that connects two sources of information:

- **Structured data** — synthetic healthcare clinical and claims datasets
- **Unstructured data** — biomedical research articles from PubMed

The AI agent can analyze both sources to provide summarized, contextual answers to user questions. :contentReference[oaicite:1]{index=1}

## Technologies Used

- **Snowflake Cortex Analyst**
- **Snowflake Cortex Search**
- **Snowflake Intelligence**
- **Semantic Views**
- **Retrieval-Augmented Generation (RAG)**

These tools enable the agent to translate natural language queries into SQL for structured data and retrieve relevant information from research documents.

## Example Capabilities

The AI agent built in this project can answer questions such as:

- *Which allergies are potentially life threatening?*
- *What are clinical strategies for latex exposure?*

It does this by combining results from Snowflake datasets with summarized research insights. :contentReference[oaicite:2]{index=2}

## Purpose

This repository serves as a reference for the **Snowflake AI agent tutorial project**, demonstrating how modern enterprise data platforms integrate:

- Data engineering  
- Large language models

- Conversational analytics
